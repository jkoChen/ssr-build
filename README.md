# ssr 服务器一键搭建脚本

clone 本项目
```
git clone https://github.com/jkoChen/ssr-build.git
```
进入文件夹
```
cd ssr-build
```
增加运行权限
```
chmod +x bbr-install.sh build.sh link_info.sh
```

安装bbr加速
```
chmod +x bbr-install.sh
./bbr-install.sh
```
安装ssr
```
chmod +x build.sh
./build.sh
```

查看链接的用户
```
chmod +x link_info.sh
./link_info.sh
```
